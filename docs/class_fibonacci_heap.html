Detailed Description

the class provides fibonacci heap. The heap works similar to binomial heap but in addition to binomial heap provides the API for arbitrary node removal along with Increase Key of an arbitrary for Max Heap and Decrease Key for an arbitrary node in a Min Heap.

Note
the data structue does not own the nodes and so it is the responsibility of the user to free nodes stored in the
